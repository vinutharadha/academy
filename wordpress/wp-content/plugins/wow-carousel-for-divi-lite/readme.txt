== Description ==
[Wow Divi Carousel Lite](https://wowcarousel.com/) is a Divi touch-enabled plugin that lets you create a beautiful responsive slider or carousel. Every heading, text, image can be edited easily with the customization options. Change all the details that really matter in typography and style. It's mobile responsive, your site can be enjoyed in any environment, be it a mobile, tablet or desktop. No tech skills needed, just pick a module form divi builder. It works like magic!

== Installation ==
Note: This plugin works with Divi Theme or Divi Builder. Make sure you have Divi Theme or Divi Builder installed.

1. Upload the plugin folder to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress
